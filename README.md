# Todo App
[Demo](https://olya-naz.github.io/todo-app/)

## Built With
  * React JS
  * LocalStorage
  * Styled components
  
## Install
Clone the repo to your local storage, and install the dependencies using NPM
<pre>
git clone https://github.com/olya-naz/todo-app.git
npm install
</pre>

## Run
<pre>npm start</pre>

## Deploy
<pre>npm run build</pre>
